Theoretical Design

I would like to introduce a novel robust control method called Combined Sliding Mode Control, to handle mismatched uncertainy in system.

Combined Sliding Mode Control

The CSMC has two sliding mode surface, one handling nominal system and the other one handling uncertainty. Details will be shown in publications Download.
It is proved in simulation test that CSMC has strong robusness to uncertainty and disturbance. Results are shown in the followings.
